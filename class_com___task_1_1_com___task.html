Detailed Description

Com_Task handles all interactions with the Bluetooth modele in a single loop.

This happens in three steps. First, all data is extracted from the UART object and stored in a local buffer. Second, one command is parsed from the buffer. Third, data is passed from the dataQueue object to the bluetooth module.
