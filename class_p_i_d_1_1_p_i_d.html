Detailed Description

PID is an generic, abstract PID class used in various places in Romi's firmware.

It is used to store constants, and when an output is needed the function calculate(error) is called. This class also features integral saturation: if the integral effort reaches an assigned threshold, the integral stops growing. In addition, the class also features the ability to average the last several derivative values for some basic filtering.
